## 📌 Requisitos

### Servidor Web
- PHP 7.4 ou superior
- MySQL 5.7 ou MariaDB 10.0+
- Apache/Nginx
- Extensões PHP: `curl`, `mysqli`, `json`, `session`

### Contas Necessárias
- Conta no [Mercado Pago](https://www.mercadopago.com.br)
- Acesso ao cPanel ou FTP do servidor
- phpMyAdmin ou acesso ao banco de dados
